Reconis

OSINT & Vulnerability Reconnaissance Platform

OSINT Python FastAPI Next.js Docker Nuclei

A professional, modular, and fully automated OSINT and vulnerability reconnaissance platform. It automates subdomain enumeration, service fingerprinting, screenshot capture, directory fuzzing, domain takeover detection, and vulnerability correlation — producing executive-ready PDF reports.

// Architecture

🖥️ Frontend

  • → Next.js (App Router)
  • → TypeScript
  • → Tailwind CSS Dashboard
  • → Real-time WebSockets

⚙️ Backend

  • → FastAPI + Uvicorn
  • → Playwright (Headless Chrome)
  • → SQLite Database
  • → WeasyPrint PDF Engine

// Key Features

🔍

Passive Recon — Subdomain harvesting & historical endpoint leaks via Wayback Machine

⚡

Async Active Scanning — Parallelized Nmap port scanning & HTTP header fingerprinting

📸

Visual Recon — Auto-screenshots of live subdomains using headless Playwright

🛡️

Takeover Detection — Identifies dangling DNS CNAMEs & fuzzes subdomain paths

🎯

Vuln Intelligence — CVE correlation via NVD + Nuclei template-based scanning

📊

Executive Reports — Auto-generated PDF reports with Matplotlib charts
